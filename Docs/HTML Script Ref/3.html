3. Client Callbacks

The following callbacks are only called when running on the client.  If any callback fails, the error message is only displayed if enableAlerts(1) was executed previously.

  1. onTeamSelect()
    Called before the player selects a team
     
  2. onFleetSelected( fleetId )
    Called after the player selects a fleet, the fleet ID is passed
     
  3. onShipSelect()
    Called before the player selects a ship
     
  4. onShipSelected( shipName )
    Called after the player selects a ship
     
  5. onShipAttached( shipId )
    players ship is spawned, so i.e. entered through a (intersystem) jumpgate, from a shipyard etc
     
  6. onShipDetached( shipId )
    players ship is removed from the universe, so i.e. entered a (intersystem) jumpgate, or a shipyard etc
     
  7. onTactical()
     
  8. onDeath( shipId, killerId )
    Called when the players ship is destroyed.  shipId is the (now destroyed) players ship, killerId is the killers Id, or nil if no killer exists.
     
  9. onCaptured( shipId, capturerId )
    Called when the players ship is captured.  shipId is the (now captured) players ship, capturerId is the capturers Id. 
     
  10. onEndGame()
    Called when the scenario or mission is completed.
