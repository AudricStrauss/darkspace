7. Ship Functions


  1. isShip( id )
    This function, like isNoun(), returns true or false, or nil on error if the object is a ship.

  2. isAIControlled( shipId )*
    Tells whether a ship is AI controlled. Returns 1 if true, nil if false.

  3. isShipDestroyed( id )
    This function, like isNoun looks for the named object, if it is a ship that is destroyed but has not yet been removed (its still in the exploding animation stage), this function will return true.  If the ship is still alive, false is returned.  If the id is not a ship, or there is some other error, nil is returned.
     
  4. shipCount()
    This function returns the total number of ships in the current system.
    local ships = shipCount();
     
  5. getShip( n )
    This function returns the key of ship n in the current system, or nil if there is an error (no such ship, etc).
    local key = getShip( 1 );
     
  6. getHull(id)
    This function returns the hull percentage of a ship.
    local h = getHull( "Cadet" );
     
  7. setHull(id)
    This function is used to set the hull of the specified ship. Returns true if hull is changed, returns nil if object not found or object is not a ship.
    -- set the hull to 50%
    SetHull( "Cadet", 50 );
     
  8. getSignature(id)
    This function returns the current signature of a ship, or nil on error.
    local s = getSignature( "Cadet" );
     
  9. getEnergy(shipId)
    This function returns the percentage of total energy the ship currently has.
     
  10. useGadget(id, key, target, shift )
    This function will use a gadget on the specified ship. Returns the number of gadgets that were used, or nil on error.
    -- useGadget( ship, key, target, shift )
    useGadget( "Cadet", "K", nil, 0 );
     
  11. playersShip()
    This function returns the name of the players ship, if any.  This should be used clientside only.
    ship = playerShip();
    pushChat( "Welcome to DarkSpace "..ship );

  12. getShipType(id)
    This function returns the type of ship as a number value, the chart below maps the numbers to the actual classes of ships in DarkSpace.
    local shipType = getShipType( "Cadet" );
    0 UNKNOWN
    1 FIGHTER
    2 ENGINEER
    3 SUPPLY
    4 TRANSPORT
    5 SCOUT
    6 FRIGATE
    7 DESTROYER
    8 CRUISER
    9 DREAD
    10 STATION

     
  13. spawnShip( resource, where, name, team )*
    This function creates a noun for the specified ship in the context. "where" can be a spawn point or a planet or other object that is currently in-game.
    -- spawnShip( resource, where, name, team )
    local key = SpawnShip( "ships/ICC/Engineering/MR 110/SC_MR110", "Spawn1", "Eng Alpha", 1 );
     
  14. orderShip(id, target, order )*
    This function sets the order for the specified ship.  This function returns true if it succeeds, and nil on errors.
    -- orderShip( name, target, order )
    orderShip( "Eng Alpha", "Mycopia", 2 );
    Order Value
    NOORDER 0
    ATTACK 1
    DEFEND 2
    CAPTURE 3
    MOVE 4
    RELOAD 5
    BEACON 6
    HOLD 7

     
  15. isJumping( id )
    This function returns true if the ship is currently in jump, false if not, or nil on errors.
     
  16. hasMission( id )
    This function returns a value of true if the ship currently has a mission. The value is false if the ship currently does not have a mission, and nil on errors.
     
  17. getShipOrder( shipId )
    Returns the mission a ship has, or nil on errors. Numbers for mission types correspond to those used for orderShip().
     
  18. enableShipAI( shipId, boolean )
    When true, this function enables high/low level ship AI.  If the boolean value is false, the ship is "dumb."
     
  19. controlShipAbsolute( shipId, heading, velocity )
    Sets a ship's absolute heading and absolute velocity.  The ship will start to turn and accelerate to match the given numbers.
     
  20. controlShipRelative( shipId, heading, velocity )
    Sets a ship's yaw and absolute velocity.  The ship will start to turn and accelerate to match the given numbers.
     
  21. getShipVelocity ( shipId )
    Returns the velocity of the given ship.
     
  22. getShipHeading ( shipId )
    Returns the heading of the given ship.
     
  23. getShipYaw ( shipId )
    Returns the yaw of the given ship.
     
  24. getAllNounsRelative ( shipId )
    returns a table 1 .. n, containing tables with the index key, x, z, vel of objects within range. Coordinates are relative to the ship and its heading, vel is the relative velocity to shipId.
    Example: An object has a distance of 50 gu to the ship. If its in front of it then z=50 and x=0, if its behind it z=-50 and x=0, if its right of it z=0 and x=50, if its left of it z=0 and x=-50.
