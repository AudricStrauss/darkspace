8. Gadget Functions


  1. isGadget( id )
    Returns true or false, or nil on error if the item is a gadget.
     
  2. gadgetCount( ship )
    This function returns the number of gadgets attached to the specified ship, or nil on errors.
     
  3. getGadget( ship, n )
    This function returns the key of gadget n on the specified ship, or nil on errors.
     
  4. getGadgetType( gadget )
    This function returns the type of gadget from it’s key. See the following table for the current types of gadgets.
    UKNOWN 0
    DRIVE 1
    SPECIAL 2
    SPECIAL_HEAVY 3
    JUMP_DRIVE 4
    WEAPON 5
    WEAPON_HEAVY 6
    WEAPON_BEAM 7
    WEAPON_MEDIUM_BEAM 8
    WEAPON_HEAVY_BEAM 9
    WEAPON_BOMB 10
    WEAPON_HEAVY_BOMB 11
    ARMOR 12
    FIGHTER_BAY 13
    DRONE_BAY 14
    ORB 15
    CLOAK 16
    HULL 17

     
  5. getGadgetMount( gadget )
    This function returns the facing of a gadget. See the table below.
    FRONT 0
    RIGHT 1
    LEFT 2
    BACK 3
    FULL 4

     
  6. setGadgetMount( gadget, mount )
    This function sets the mount for a gadget, based upon the above table.  This function returns true if it succeeds, and nil on errors.

  7. getGadgetAmmo( gadget )
    This function returns the amount of ammo remaining if the gadget is a weapon that requires ammo, or 0 if no ammo is used.
     
  8. setGadgetAmmo( gadget, amount )
    This function allows you to set the amount of ammo in a gadget.
     
  9. useGadget2( gadget, target, shift )
    This function will use the specified gadget on the target. This function takes a gadget key, unlike the older useGadget() function which uses a gadget based on it’s hotkey.  This function returns true if it succeeds, and nil on errors.
     
  10. isGadgetActive( gadgetId )
    Returns true if the gadget is active or false, or nil on error.  The jumpdrive will show up as active when aligning, as well as jumping; use isJumping() to check for an active jump.
     
  11. isGadgetUsable( gadgetId, shiftStatus [, targetId] )
    Returns 1 if gadget is usable, nil if false.  shitStatus = 1 if shift should be pressed, 0 if not.  targetId is an optional target; some gadgets (such as the tractor beam) need a target for the check, others (like ECM) do not.  If no target is given though one is needed, the function will return nil.
     
  12. beamPointDefenseEnabled( gadgetId )
    Returns 1 if point defense is enabled for the passed gadget, nil if it is not..
     
  13. setBeamPointDefense( gadgetId, enable )
    This function lets you set the point defense on a gadget via the enable variable: 1 enables PD, 0 disables it.  This function returns true if it succeeds, and nil on errors.
     
  14. getJumpTarget( gadgetId, enable )
    This function returns the jumptarget of the jumpdrive.  If there is no target, nil is returned.
     
  15. hasGadgetAmmo( gadgetId )
    Returns true if the gadget uses ammo for operation.
     
  16. getShielding( gadgetId )
    Returns the current strength of the shield or armor given, in percent.
     
  17. setShielding( gadgetId, percent )
    This allows you to set the strength of a shield or armor gadget in percent.
